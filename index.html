NeosMemo Commands

Navigate to Script directory:

cd C:/Users/epuser/Desktop/Script
ls

Activate the virtual environment:

.\myenv\Scripts\Activate

Login to OpenMaint server:

ssh epadmin@192.168.9.232
Ep492033

Continue to Docker:

sudo docker exec -it 825bb6beb037 bash
Ep492033

Continue to psql:

psql -U postgres -d openmaint

Run the command in psql to copy and retrieve the data:

\copy (SELECT t1."Number", t1."ShortDescr", t1."OpeningDate", t1."ClosureDate", t2."Description", t3."Description", t4."Description", t1."Priority", t1."ProcessStatus", t1."ProcessNotes" FROM "CorrectiveMaint" t1 INNER JOIN "Room" t2 ON t1."Room" = t2."CurrentId" INNER JOIN "Site" t3 ON t1."Site" = t3."CurrentId" INNER JOIN "Team" t4 ON t1."Team" = t4."CurrentId" WHERE "ProcessNotes" IS NOT NULL) TO '/home/pro_notes4.csv' WITH CSV HEADER;

Exit psql:

\q

Exit Docker:

exit

Extract data from Docker to VM:

sudo docker cp openmaint_db:/home/pro_notes4.csv /home/epadmin

Exit VM to your PC:

exit

Copy the data from VM to your PC:

scp -r epadmin@192.168.9.232:/home/epadmin/pro_notes2.csv C:/Users/epuser/Desktop/Script

python Conversion_Script.py
